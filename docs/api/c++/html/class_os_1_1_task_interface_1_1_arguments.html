construct a set of arguments to start a task

Construct a set of arguments to start a task. It is illegal to supply a task routine that is set to a nullptr.

Parameters
namename of the task
routineroutine to run as part of this task
routine_argument(optional) argument to supply to the task routine
priority(optional) priority of this task
stackSize(optional) size of stack supplied to this task
cpuAffinity(optional) cpu affinity of this task. TODO: fix this into an array
identifier(optional) identifier for this task
