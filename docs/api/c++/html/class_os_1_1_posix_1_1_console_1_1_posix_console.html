Detailed Description

posix implementation of Os::ConsoleInterface

Posix implementation of ConsoleInterface for use as a delegate class handling posix console operations. Posix consoles write to either standard out or standard error. The default file descriptor used is standard out. This may be changed by calling setOutputStream.
